**1. Create apps/web/lib/scene-utils.ts (per D-05, D-07):**

Export a pure function `detectCollapsibleRanges` that identifies runs of consecutive agent messages that should collapse.

```typescript
import type { HubEventEnvelope } from '@beagle-console/shared';

export interface CollapsibleRange {
  startSeq: number;       // first sequence number in the range
  endSeq: number;         // last sequence number in the range
  sequences: number[];    // all sequence numbers in this range
  agentIds: string[];     // unique agent IDs involved (deduplicated)
  messageCount: number;
  startTime: string;      // ISO timestamp of first message
  endTime: string;        // ISO timestamp of last message
}

// NEVER-COLLAPSE types per D-07
const UNCOLLAPSIBLE_TYPES: Set<string> = new Set([
  'plan_proposal', 'question', 'artifact', 'state_transition', 'tldr_update',
]);

export function detectCollapsibleRanges(
  eventOrder: number[],
  events: Record<number, HubEventEnvelope>,
): CollapsibleRange[] {
  // Algorithm:
  // Walk eventOrder. Track consecutive agent_message events where agentId !== 'user'.
  // When a user message OR uncollapsible event type is encountered, end the current run.
  // If the run has >3 messages (per D-05), emit a CollapsibleRange.
  // Return array of ranges.
}
```

Implementation details:
- A "run" starts when we see an `agent_message` where `agentId !== 'user'`.
- A "run" ends when we hit: (a) a user message (agentId === 'user'), (b) an event with type in UNCOLLAPSIBLE_TYPES, (c) end of list.
- Only runs with >3 messages become CollapsibleRange (per D-05: ">3 consecutive messages").
- `agentIds` = deduplicated set of agentIds in the run, sorted alphabetically.
- The function is pure — no store dependency. It receives data as args.
