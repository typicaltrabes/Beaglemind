## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-08-06 | Spoofing | token brute-force | mitigate | 256-bit random token (32 bytes hex) makes enumeration infeasible |
| T-08-07 | Denial of Service | replay endpoint | accept | Rate limiting deferred to Claude's discretion; Year 1 low traffic |
| T-08-08 | Information Disclosure | tenant iteration | mitigate | Token lookup iterates tenants server-side; tenant IDs never exposed to client |
| T-08-09 | Repudiation | replay access | mitigate | Every view logged with IP, user-agent, timestamp in replay_views table |
</threat_model>
