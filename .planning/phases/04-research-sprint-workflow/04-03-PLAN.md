**1. SSE Endpoint: apps/web/app/api/runs/[id]/stream/route.ts** (per D-16, D-17)

Set `export const runtime = 'nodejs';` and `export const dynamic = 'force-dynamic';`

GET handler:
- `const { tenantId } = await requireTenantContext();`
- `const { id: runId } = await params;`
- Read `Last-Event-ID` from request headers for reconnection: `const lastEventId = request.headers.get('Last-Event-ID');`
- Create `ReadableStream` with `start(controller)`:
  - `let closed = false;`
  - Helper `send(eventId, data)`: enqueue `id: ${eventId}\ndata: ${data}\n\n` (SSE format)
  - Define async `replay()` function (do NOT await in start):
    - Get tenant DB: `const { db: tdb, schema } = getTenantDb(tenantId);`
    - Import `eq, asc, gt` from `drizzle-orm`
    - Query events: if lastEventId, `where(and(eq(events.runId, runId), gt(events.sequenceNumber, parseInt(lastEventId))))`, else `where(eq(events.runId, runId))`, ordered by `asc(events.sequenceNumber)`
    - For each event, call `send(String(event.sequenceNumber), JSON.stringify(event))`
    - Create Redis subscriber: `const Redis = (await import('ioredis')).default; const subscriber = new Redis(process.env.REDIS_URL ?? 'redis://localhost:6379');`
    - Subscribe to channel `run:${tenantId}:${runId}`
    - On 'message': parse JSON, call `send(String(parsed.sequenceNumber), message)`
    - Set up heartbeat: `setInterval(() => { if (!closed) controller.enqueue(encoder.encode(': heartbeat\n\n')); }, 30000)`
    - Set up abort listener on `request.signal`: set closed=true, clearInterval heartbeat, unsubscribe+quit Redis, try controller.close()
  - Call `replay().catch(err => { console.error('SSE error:', err); try { controller.close(); } catch {} })`
  - Do NOT await replay()

Return `new Response(stream, { headers: { 'Content-Type': 'text/event-stream', 'Cache-Control': 'no-cache, no-transform', 'Connection': 'keep-alive', 'X-Accel-Buffering': 'no' } })`
