  2. `{ from: 'approved', to: 'executing' }` with triggeredBy = 'system' in content
- Return `{ ok: true }`

**2. handleQuestionAnswer** (per D-15):

Add Zod schema:
```typescript
const QuestionAnswerBody = z.object({
  runId: z.string().uuid(),
  tenantId: z.string().uuid(),
  questionId: z.string().uuid(),
  answer: z.string(),
  agentId: z.string(), // which agent asked the question
});
```

Add handler function `handleQuestionAnswer(body, registry, router)`:
- Parse body with QuestionAnswerBody
- Construct OpenClawOutbound message to the asking agent: content = `[ANSWER to Q-{questionId}] {answer}`, senderId = `console_{tenantId}`, customData = `{ runId, tenantId, questionId }`
- Send to asking agent via `registry.send(parsed.agentId, outbound)`
- Persist an agent_message event: type='agent_message', agentId='user', content = `{ text: answer, questionId, isAnswer: true }`
- Return `{ ok: true }`

Remove the `persistAndPublish` call for `state_transition` from `handleRunStart`. Keep everything else (setActiveRun, sending prompt to agent).

Export all handlers: `handleSend`, `handleRunStart`, `handleRunStop`, `handleRunApprove`, `handleQuestionAnswer`.
