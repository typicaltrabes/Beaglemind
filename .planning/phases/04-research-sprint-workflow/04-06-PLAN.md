**1. MessageList: apps/web/components/transcript/message-list.tsx** (per D-21)

'use client' component.

Props: `{ runId: string }`

Reads events from Zustand: `const events = useRunStore((s) => s.eventOrder);` and `const eventMap = useRunStore((s) => s.events);`

Renders a scrollable list (use ScrollArea from shadcn) that maps over eventOrder and renders each event based on type:

```typescript
function renderEvent(event: HubEventEnvelope, runId: string) {
  switch (event.type) {
    case 'plan_proposal':
      return <PlanCard event={event} runId={runId} />;
    case 'question':
      return <QuestionCard event={event} runId={runId} />;
    case 'artifact':
      return <ArtifactCard event={event} />;
    case 'agent_message':
      return <AgentMessage event={event} />;
    case 'state_transition':
      return <StateTransitionMessage event={event} />;
    case 'system':
      return null; // Don't render system events (typing, etc.)
    default:
      return null;
  }
}
```

Inline sub-components (can be in same file):

**AgentMessage**: Shows agent name (event.agentId) with color coding (Mo=amber-500, Jarvis=teal-500, user=blue-500, default=gray-400), timestamp, and message text (event.content.text as string). Simple div with agent name label + content paragraph.

**StateTransitionMessage**: Small system-style message: "Run transitioned from {from} to {to}" in muted text, centered, with timestamp.

Auto-scroll: use a `useRef` on the scroll container and scroll to bottom when new events arrive (useEffect on eventOrder.length).

Empty state: "Waiting for events..." when no events.
